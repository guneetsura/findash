# Additional Feature Scope for Future

Below are some enhancements to consider for future scope.

## Planned Features

### Data & Analytics

- Budget tracking with category limits & alerts
- Savings goals progress tracker
- Spending trends visualization over time (week-over-week comparison)
- Net worth calculator
- Income vs expense breakdown by month (projection)

### User Experience

- Recurring transactions (auto-add monthly bills)
- Transaction notes/tags
- Search suggestions and recent search history
- Custom categories — Create, edit, and delete transaction categories with custom names and colors
- Bulk transaction actions (delete multiple, categorize batch)
- Transaction history/undo functionality

### Visualization

- Calendar heatmap of daily spending
- Comparison charts (this month vs last month vs avg)
- Trend line projections for spending patterns
- Category distribution by income/expense ratios

### Security & Data

- Multiple accounts/wallets support
- Data import (CSV upload with validation)
- Export filtered transactions (subset by date range, category)
- Data backup and restore functionality
- Cloud sync option

### Integrations & Customization

- Currency converter and multi-currency support
- Theme customizer (preset + custom color editor)
- Bank account linking (mock or real API)
- Expense categorization suggestions (AI)
- Receipt upload and OCR parsing
